The density path is Gaussian smoothing

At the distribution level, the rescaled corruption model gives a concrete path $$p_\sigma = \operatorname{Law}(X+\sigma Z)= p * \mathcal{N}(0,\sigma^2 I).$$

Convolution path $$p_\sigma = \operatorname{Law}(X+\sigma Z)= p * \mathcal{N}(0,\sigma^2 I)$$ As $\sigma$ grows, fine structure is washed out and nearby modes start to blend together.
Practical start point There is no literal \(\mathcal{N}(0,\infty^2 I)\), so in practice we stop at a large finite \(\sigma_{\max}\), often around \(80\), and use $$p_{\sigma_{\max}} \approx \mathcal{N}(\mu,\sigma_{\max}^2 I), \qquad \mu=\mathbb{E}[X].$$
The point is conceptual, not exact numerics: sigma indexes a family of smoothed distributions. In practice we never start from infinite variance; we start from a large finite sigma_max and treat that endpoint as approximately Gaussian.
