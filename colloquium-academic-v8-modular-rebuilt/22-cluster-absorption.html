Intermediate stage: local clusters

Cluster absorption (Prop. 6.9, informal) Suppose one cluster \(S\subset \mathrm{supp}(p)\) has diameter \(D\), and the rest of the data stays at distance at least \(2D\) from \(\mathrm{conv}(S)\). For every \(\epsilon>0\) there is an explicit \(\sigma_0(S,\epsilon) > 0\) such that if \(\sigma_1 < \sigma_0\) and \(x_{\sigma_1}\in \overline{B_{D/2-\epsilon}(S)}\), then the trajectory never leaves \(\overline{B_{D/2-\epsilon}(S)}\) and converges into \(\mathrm{conv}(S)\) as \(\sigma\to 0\).
Use the previous-slide cluster geometry visually so the audience can literally see what the separated cluster S is. Emphasize that the absorbing set is a tube of radius just under D/2, so the cluster really holds the trajectory once the noise is small enough.
